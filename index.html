Bing / IndexNow 建議

  1. 維持 /sitemap.xml 與 /robots.txt，在 Bing Webmaster Tools 提交站點。
  2. 啟用 IndexNow：在網域根目錄放置 key 檔，並於新增/更新頁面時 POST URL 至 IndexNow 端點，可顯著加速收錄。

（為何：IndexNow 由 Bing 主導，能縮短內容變更到可搜尋之間的時間差）
